Invisibyte – Secure PDF Image Steganography Tool
● Developed an image steganography system to securely hide PDF files using LSB encoding.
● Implemented password-based encryption and decryption with support for alphanumeric and special characters.
● Built a Tkinter-based GUI with clean design, intuitive navigation, and options to save encoded images and decode hidden
PDF files
● Designed for educational use to securely transmit exam papers via encrypted images.
● Tech Stack: Python, Tkinter, PIL, OpenCV, cryptography
